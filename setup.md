What
====

A crazy simple demo of zeromq and keyczar for encrypted (shared secret) data exchange over 0mq

Installation
============

* yum install zeromq zeromq-devel
* pip install pyzmq
* pip install pyasn1 
* pip install PyCrypto
* pip install python-keyczar
